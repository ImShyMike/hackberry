How can I participate?

  1. Read the rules
  2. Subscribe to the project
  3. Install Hackatime (tutorial HERE)
  4. Develop your project for at least 30 hours (remember to commit on GitHub frequently!)
  5. Upload your project to GitHub, adding a description and including a short video where you showcase it
  6. Share the GitHub link in the #HackBerry-ship channel on Slack
  7. Enter your shipping details HERE

Rules!

    NO Autokey or time cheat (fraud will ban you from all HC!)

    NO Copy & paste of other people's project

    NO Old projects or used in other HC programs (except SOM*)

    YES, you can use tutorials, but you must add some personalization

    YES, you can use AI, but only as a support!
